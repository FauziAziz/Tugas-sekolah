1. Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web.

HTML menggunakan tag untuk menandai dan mengatur struktur konten pada halaman web.

2. Struktur Dasar HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <!-- Konten di sini -->
</body>
</html>

3. Tag HTML Dasar

Heading (Judul):

<h1>Heading 1 - Paling Besar</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6 - Paling Kecil</h6>

Paragraf:

<p>Ini adalah paragraf.</p>

Line Break:

Baris pertama<br>
Baris kedua

Horizontal Rule:

<hr>

4. Text Formatting

  • <strong> atau <b> - teks tebal
  • <em> atau <i> - teks miring
  • <u> - teks bergaris bawah
  • <mark> - teks di-highlight
  • <small> - teks kecil
  • <del> - teks dicoret
  • <ins> - teks yang disisipkan
  • <sub> - subscript (H₂O)
  • <sup> - superscript (x²)

Contoh:

<p>Ini <strong>teks tebal</strong> dan <em>teks miring</em></p>

5. Link (Hyperlink)

<a href="https://google.com">Link ke Google</a>
<a href="halaman.html">Link ke halaman lain</a>
<a href="#bagian">Link ke bagian di halaman yang sama</a>
<a href="mailto:email@email.com">Link Email</a>
<a href="tel:+628123456789">Link Telepon</a>

<!-- Target -->
<a href="url" target="_blank">Buka di tab baru</a>
<a href="url" target="_self">Buka di tab yang sama</a>

6. Image (Gambar)

<img src="gambar.jpg" alt="Deskripsi gambar">
<img src="gambar.jpg" alt="Deskripsi" width="300" height="200">

<!-- Gambar dengan link -->
<a href="url">
    <img src="gambar.jpg" alt="Deskripsi">
</a>

9. Form

<form action="proses.php" method="post">
    <!-- Input Text -->
    <label for="nama">Nama:</label>
    <input type="text" id="nama" name="nama" required>
    
    <!-- Input Email -->
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <!-- Input Password -->
    <label for="password">Password:</label>
    <input type="password" id="password" name="password">
    
    <!-- Input Number -->
    <label for="umur">Umur:</label>
    <input type="number" id="umur" name="umur" min="1" max="100">
    
    <!-- Input Date -->
    <label for="tanggal">Tanggal:</label>
    <input type="date" id="tanggal" name="tanggal">
    
    <!-- Radio Button -->
    <label>Jenis Kelamin:</label>
    <input type="radio" id="laki" name="gender" value="laki">
    <label for="laki">Laki-laki</label>
    <input type="radio" id="perempuan" name="gender" value="perempuan">
    <label for="perempuan">Perempuan</label>
    
    <!-- Checkbox -->
    <input type="checkbox" id="setuju" name="setuju">
    <label for="setuju">Saya setuju</label>
    
    <!-- Select Dropdown -->
    <label for="kota">Kota:</label>
    <select id="kota" name="kota">
        <option value="">Pilih Kota</option>
        <option value="jakarta">Jakarta</option>
        <option value="bandung">Bandung</option>
        <option value="surabaya">Surabaya</option>
    </select>
    
    <!-- Textarea -->
    <label for="pesan">Pesan:</label>
    <textarea id="pesan" name="pesan" rows="4" cols="50"></textarea>
    
    <!-- File Upload -->
    <label for="file">Upload File:</label>
    <input type="file" id="file" name="file">
    
    <!-- Button -->
    <button type="submit">Kirim</button>
    <button type="reset">Reset</button>
</form>

10. Semantic HTML

<header>Header/Kepala halaman</header>
<nav>Navigasi menu</nav>
<main>Konten utama</main>
<article>Artikel atau konten mandiri</article>
<section>Bagian/seksi konten</section>
<aside>Konten samping/sidebar</aside>
<footer>Footer/Kaki halaman</footer>

Contoh Struktur:

<!DOCTYPE html>
<html>
<head>
    <title>Website</title>
</head>
<body>
    <header>
        <h1>Judul Website</h1>
        <nav>
            <ul>
                <li><a href="#">Home</a></li>
                <li><a href="#">About</a></li>
                <li><a href="#">Contact</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <h2>Judul Artikel</h2>
            <p>Konten artikel...</p>
        </article>
        
        <aside>
            <h3>Sidebar</h3>
            <p>Konten samping...</p>
        </aside>
    </main>
    
    <footer>
        <p>© 2025 Website Saya</p>
    </footer>
</body>
</html>

13. Meta Tags

<head>
    <!-- Charset -->
    <meta charset="UTF-8">
    
    <!-- Viewport untuk responsive -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- Deskripsi untuk SEO -->
    <meta name="description" content="Deskripsi website">
    
    <!-- Keywords -->
    <meta name="keywords" content="html, css, javascript">
    
    <!-- Author -->
    <meta name="author" content="Nama Anda">
    
    <!-- Refresh otomatis -->
    <meta http-equiv="refresh" content="30">
</head>

14. HTML Entities

  • &lt; - kurang dari (<)
  • &gt; - lebih dari (>)
  • &amp; - ampersand (&)
  • &quot; - tanda kutip (")
  • &apos; - apostrof (')
  • &copy; - copyright (©)
  • &reg; - registered (®)
  • &nbsp; - non-breaking space

15. HTML Atribut Global

  • id - identifier unik
  • class - untuk styling CSS
  • style - inline CSS
  • title - tooltip saat hover
  • lang - bahasa konten
  • hidden - menyembunyikan elemen
  • data-* - custom data attributes

Contoh:

<div id="main" class="container" title="Ini tooltip" data-user="123">
    Konten
</div>

19. Best Practices

  • Selalu gunakan DOCTYPE html
  • Gunakan semantic HTML untuk SEO yang baik
  • Tutup semua tag dengan benar
  • Gunakan lowercase untuk nama tag dan atribut
  • Selalu tambahkan atribut alt pada gambar
  • Gunakan indentasi yang konsisten
  • Validasi HTML dengan W3C Validator
  • Gunakan komentar untuk dokumentasi
  • Pisahkan konten (HTML) dari presentasi (CSS) dan behavior (JavaScript)
