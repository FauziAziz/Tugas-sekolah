Materi Lengkap HTML

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. HTML bukanlah bahasa pemrograman, melainkan bahasa markup yang menggunakan tag untuk menandai elemen-elemen di halaman web.

HTML digunakan untuk:

  • Membuat struktur dasar halaman web
  • Menyusun konten (teks, gambar, video)
  • Membuat link antar halaman
  • Membangun formulir interaktif

Struktur Dasar HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <!-- Konten halaman web -->
</body>
</html>

Penjelasan Struktur:

  • <!DOCTYPE html>: Deklarasi tipe dokumen HTML5
  • <html>: Elemen root dari dokumen HTML
  • <head>: Berisi metadata (informasi tentang dokumen)
  • <meta charset="UTF-8">: Menentukan encoding karakter
  • <title>: Judul halaman yang muncul di tab browser
  • <body>: Berisi semua konten yang terlihat di halaman web

Elemen-elemen HTML Penting

1. Elemen Teks

Tag Fungsi Contoh
<h1> - <h6> Heading (judul) <h1>Judul Utama</h1>
<p> Paragraf <p>Ini paragraf</p>
<strong> Teks tebal (penting) <strong>Penting</strong>
<em> Teks miring (penekanan) <em>Penekanan</em>
<br> Line break (baris baru) Teks 1<br>Teks 2

2. Elemen List

Ordered List (Berurutan):

<ol>
    <li>Item pertama</li>
    <li>Item kedua</li>
    <li>Item ketiga</li>
</ol>

Unordered List (Tidak Berurutan):

<ul>
    <li>Item A</li>
    <li>Item B</li>
    <li>Item C</li>
</ul>

3. Elemen Link dan Gambar

Link:

<a href="https://example.com">Kunjungi Website</a>

Gambar:

<img src="gambar.jpg" alt="Deskripsi gambar" width="300">

4. Elemen Formulir

Formulir digunakan untuk mengumpulkan data dari pengguna:

<form action="proses.php" method="post">
    <label for="nama">Nama:</label>
    <input type="text" id="nama" name="nama">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <label for="password">Password:</label>
    <input type="password" id="password" name="password">
    
    <button type="submit">Kirim</button>
</form>

5. Elemen Tabel

<table>
    <thead>
        <tr>
            <th>Nama</th>
            <th>Usia</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Andi</td>
            <td>25</td>
        </tr>
        <tr>
            <td>Budi</td>
            <td>30</td>
        </tr>
    </tbody>
</table>

HTML5 Semantic Elements

HTML5 memperkenalkan elemen semantik yang memberikan makna pada struktur halaman:

  • <header>: Bagian header halaman
  • <nav>: Navigasi
  • <main>: Konten utama
  • <section>: Bagian konten
  • <article>: Konten mandiri
  • <aside>: Konten samping (sidebar)
  • <footer>: Bagian footer

Atribut HTML

Atribut memberikan informasi tambahan tentang elemen:

Atribut Deskripsi Contoh
id Identifier unik elemen <div id="header">
class Nama kelas untuk styling <p class="intro">
src Sumber file (gambar, video) <img src="logo.png">
href URL link <a href="page.html">
alt Teks alternatif gambar <img alt="Logo">

Validasi HTML

Validasi HTML penting untuk:

  • Memastikan kode sesuai standar W3C
  • Meningkatkan kompatibilitas browser
  • Meningkatkan SEO
  • Membantu aksesibilitas

Alat validasi: W3C Markup Validation Service

Kesimpulan

HTML adalah fondasi dari semua halaman web. Dengan memahami:

  • Struktur dasar dokumen HTML
  • Elemen-elemen penting dan penggunaannya
  • Atribut untuk memberikan informasi tambahan
  • Elemen semantik HTML5 untuk struktur yang baik
  • Praktik terbaik dalam penulisan HTML

Anda dapat membuat halaman web yang terstruktur dengan baik dan siap untuk diberi gaya dengan CSS dan fungsionalitas dengan JavaScript atau PHP.
