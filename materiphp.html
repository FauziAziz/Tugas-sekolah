Materi PHP

Oleh: Nama Penulis • 19 November 2025 • 5 menit baca

1. Pengenalan PHP

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk membuat website dinamis.

File PHP menggunakan ekstensi .php dan dapat berisi HTML, CSS, JavaScript, dan kode PHP.

2. Syntax Dasar

Kode PHP ditulis di antara tag pembuka dan penutup:

<?php
    echo "Hello World!";
?>

3. Variabel

Variabel di PHP diawali dengan tanda $:

<?php
    $nama = "Budi";
    $umur = 25;
    $tinggi = 170.5;
    $aktif = true;
?>

4. Tipe Data

  • String - teks dalam tanda petik
  • Integer - bilangan bulat
  • Float - bilangan desimal
  • Boolean - true/false
  • Array - kumpulan data
  • Object - instance dari class

5. Operator

Operator Aritmatika:

  • + (penjumlahan)
  • - (pengurangan)
  • * (perkalian)
  • / (pembagian)
  • % (modulus/sisa bagi)

Operator Perbandingan:

  • == (sama dengan)
  • != (tidak sama dengan)
  • &
