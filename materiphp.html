Materi PHP Terpilih

Apa itu PHP?

PHP (PHP: Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk pengembangan web. PHP dapat disisipkan ke dalam HTML dan dijalankan di server, menghasilkan HTML yang dikirim ke browser.

Tag PHP

PHP diawali dengan <?php dan diakhiri dengan ?>:

<?php
    // Kode PHP di sini
    echo "Hello, World!";
?>

PHP dapat disisipkan di mana saja dalam HTML:

<!DOCTYPE html>
<html>
<body>
    <h1><?php echo "Judul Dinamis"; ?></h1>
    <p>Tanggal hari ini: <?php echo date('Y-m-d'); ?></p>
</body>
</html>

Tipe Data

Tipe Contoh Deskripsi
String "Hello" Teks
Integer 42 Bilangan bulat
Float 3.14 Bilangan desimal
Boolean true Nilai benar/salah
Array [1, 2, 3] Koleksi nilai
NULL null Tanpa nilai

Aritmatika

Operator aritmatika dalam PHP:

Operator Deskripsi Contoh
+ Penjumlahan $a + $b
- Pengurangan $a - $b
* Perkalian $a * $b
/ Pembagian $a / $b
% Modulus (sisa bagi) $a % $b
** Pangkat $a ** $b 
<?php
    $a = 10;
    $b = 3;
    
    echo $a + $b; // 13
    echo $a - $b; // 7
    echo $a * $b; // 30
    echo $a / $b; // 3.333...
    echo $a % $b; // 1
    echo $a ** $b; // 1000
?>

Percabangan

IF - ELSEIF - ELSE

<?php
    $nilai = 85;
    
    if ($nilai >= 90) {
        echo "A";
    } elseif ($nilai >= 80) {
        echo "B";
    } elseif ($nilai >= 70) {
        echo "C";
    } else {
        echo "D";
    }
?>

SWITCH

<?php
    $grade = "B";
    
    switch ($grade) {
        case "A":
            echo "Sempurna";
            break;
        case "B":
            echo "Baik";
            break;
        case "C":
            echo "Cukup";
            break;
        default:
            echo "Kurang";
    }
?>

Perulangan

FOR Loop

<?php
    for ($i = 1; $i <= 5; $i++) {
        echo "Nomor: $i <br>";
    }
?>

WHILE Loop

<?php
    $i = 1;
    while ($i <= 5) {
        echo "Iterasi: $i <br>";
        $i++;
    }
?>

DO-WHILE Loop

<?php
    $i = 1;
    do {
        echo "Nomor: $i <br>";
        $i++;
    } while ($i <= 5);
?>

FOREACH Loop

<?php
    $colors = ["Merah", "Hijau", "Biru"];
    
    foreach ($colors as $color) {
        echo "Warna: $color <br>";
    }
    
    // Dengan key
    $person = ["nama" => "John", "umur" => 30];
    foreach ($person as $key => $value) {
        echo "$key: $value <br>";
    }
?>

Function

Fungsi Dasar

<?php
    function sapa($nama) {
        return "Halo, $nama!";
    }
    
    echo sapa("John"); // Halo, John!
?>

Fungsi dengan Default Parameter

<?php
    function hitungHarga($harga, $diskon = 0) {
        return $harga - ($harga * $diskon / 100);
    }
    
    echo hitungHarga(100000); // 100000
    echo hitungHarga(100000, 10); // 90000
?>

Fungsi Bawaan PHP

<?php
    // String functions
    echo strlen("Hello"); // 5
    echo str_replace("World", "PHP", "Hello World"); // Hello PHP
    
    // Array functions
    $numbers = [3, 1, 4, 2];
    sort($numbers);
    print_r($numbers); // [1, 2, 3, 4]
    
    // Math functions
    echo round(3.7); // 4
    echo rand(1, 10); // Angka random 1-10
?>

Method GET

Method GET mengirim data melalui URL:

Form HTML dengan GET

<form action="proses.php" method="get">
    Nama: <input type="text" name="nama">
    Umur: <input type="number" name="umur">
    <button type="submit">Kirim</button>
</form>

Memproses Data GET di PHP

<?php
    // proses.php
    if (isset($_GET['nama']) && isset($_GET['umur'])) {
        $nama = htmlspecialchars($_GET['nama']);
        $umur = htmlspecialchars($_GET['umur']);
        
        echo "Nama: $nama <br>";
        echo "Umur: $umur tahun";
    }
?>

Contoh URL Hasil

http://example.com/proses.php?nama=John&umur=25

Karakteristik GET

  • Data terlihat di URL
  • Tidak cocok untuk data sensitif
  • Batasan panjang URL (tergantung browser)
  • Dapat di-bookmark

OOP (Object-Oriented Programming)

Class dan Object

<?php
    class Mobil {
        // Properties
        public $merk;
        public $warna;
        private $tahun;
        
        // Constructor
        public function __construct($merk, $warna, $tahun) {
            $this->merk = $merk;
            $this->warna = $warna;
            $this->tahun = $tahun;
        }
        
        // Method
        public function getInfo() {
            return "Mobil: {$this->merk}, Warna: {$this->warna}, Tahun: {$this->tahun}";
        }
        
        // Getter
        public function getTahun() {
            return $this->tahun;
        }
    }
    
    // Membuat object
    $mobil1 = new Mobil("Toyota", "Hitam", 2020);
    echo $mobil1->getInfo(); // Mobil: Toyota, Warna: Hitam, Tahun: 2020
    echo $mobil1->getTahun(); // 2020
?>

Inheritance (Pewarisan)

<?php
    class Kendaraan {
        protected $jenis;
        
        public function __construct($jenis) {
            $this->jenis = $jenis;
        }
    }
    
    class Motor extends Kendaraan {
        private $cc;
        
        public function __construct($jenis, $cc) {
            parent::__construct($jenis);
            $this->cc = $cc;
        }
        
        public function getInfo() {
            return "Motor {$this->jenis}, {$this->cc}cc";
        }
    }
    
    $motor = new Motor("Sport", 250);
    echo $motor->getInfo(); // Motor Sport, 250cc
?>

Encapsulation

  • Public: Dapat diakses dari mana saja
  • Protected: Hanya dalam class dan child class
  • Private: Hanya dalam class itu sendiri

Polimorfisme

<?php
    interface Bentuk {
        public function hitungLuas();
    }
    
    class Persegi implements Bentuk {
        private $sisi;
        
        public function __construct($sisi) {
            $this->sisi = $sisi;
        }
        
        public function hitungLuas() {
            return $this->sisi * $this->sisi;
        }
    }
    
    class Lingkaran implements Bentuk {
        private $jari;
        
        public function __construct($jari) {
            $this->jari = $jari;
        }
        
        public function hitungLuas() {
            return pi() * $this->jari * $this->jari;
        }
    }
    
    $persegi = new Persegi(5);
    $lingkaran = new Lingkaran(7);
    
    echo "Luas Persegi: " . $persegi->hitungLuas();
    echo "Luas Lingkaran: " . round($lingkaran->hitungLuas(), 2);
?>

MVC (Model-View-Controller)

Konsep MVC

MVC adalah arsitektur pemrograman yang memisahkan aplikasi menjadi tiga komponen utama:

  1. Model: Mengelola data dan logika bisnis
  2. View: Menampilkan data ke pengguna (UI)
  3. Controller: Menghubungkan Model dan View

Struktur Folder MVC

project/
├── app/
│   ├── controllers/
│   │   └── UserController.php
│   ├── models/
│   │   └── User.php
│   └── views/
│       └── user/
│           ├── index.php
│           └── show.php
├── public/
│   └── index.php
└── config/
    └── database.php

Contoh Implementasi

1. Model (User.php)

<?php
class User {
    private $db;
    
    public function __construct($db) {
        $this->db = $db;
    }
    
    public function getAllUsers() {
        $query = "SELECT * FROM users";
        return $this->db->query($query)->fetchAll();
    }
    
    public function getUserById($id) {
        $query = "SELECT * FROM users WHERE id = ?";
        $stmt = $this->db->prepare($query);
        $stmt->execute([$id]);
        return $stmt->fetch();
    }
}
?>

2. View (index.php)

<!DOCTYPE html>
<html>
<head>
    <title>Daftar User</title>
</head>
<body>
    <h1>Daftar User</h1>
    <table border="1">
        <tr>
            <th>ID</th>
            <th>Nama</th>
            <th>Email</th>
        </tr>
        <?php foreach ($users as $user): ?>
        <tr>
            <td><?php echo $user['id']; ?></td>
            <td><?php echo $user['name']; ?></td>
            <td><?php echo $user['email']; ?></td>
        </tr>
        <?php endforeach; ?>
    </table>
</body>
</html>

3. Controller (UserController.php)

<?php
class UserController {
    private $userModel;
    
    public function __construct($db) {
        $this->userModel = new User($db);
    }
    
    public function index() {
        $users = $this->userModel->getAllUsers();
        require 'views/user/index.php';
    }
    
    public function show($id) {
        $user = $this->userModel->getUserById($id);
        require 'views/user/show.php';
    }
}
?>

Keuntungan MVC

  • Pemisahan tugas yang jelas
  • Kode lebih terorganisir
  • Mudah dalam pengembangan tim
  • Reusability kode
  • Mudah dalam maintenance

Kesimpulan

Materi PHP yang telah dibahas mencakup:

  • Tag PHP: Cara menulis kode PHP dalam HTML
  • Tipe Data: Jenis-jenis data yang dapat digunakan
  • Aritmatika: Operasi matematika dasar
  • Percabangan: Pengambilan keputusan dengan IF/SWITCH
  • Perulangan: Iterasi dengan FOR, WHILE, FOREACH
  • Function: Membuat dan menggunakan fungsi
  • Method GET: Mengirim data melalui URL
  • OOP: Pemrograman berorientasi objek
  • MVC: Arsitektur aplikasi terstruktur

Dengan memahami konsep-konsep ini, Anda dapat membangun aplikasi web yang terstruktur dan efisien menggunakan PHP.
