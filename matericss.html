Materi Lengkap CSS

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan format dokumen HTML. CSS memisahkan konten (HTML) dari presentasi (tampilan), sehingga memudahkan pengelolaan dan pemeliharaan website.

CSS digunakan untuk:

  • Mengatur warna, font, dan spacing
  • Mengatur layout dan positioning
  • Membuat desain responsif untuk berbagai device
  • Menambahkan animasi dan transisi

Cara Kerja CSS

CSS bekerja dengan aturan (rules) yang terdiri dari:

  1. Selector: Memilih elemen HTML yang akan di-style
  2. Declaration Block: Berisi satu atau lebih deklarasi
  3. Declaration: Terdiri dari property dan value
selector {
    property: value;
    property: value;
}

Penempatan CSS

1. Inline CSS

Style langsung pada elemen HTML menggunakan atribut style:

<p style="color: blue; font-size: 16px;">Teks berwarna biru</p>

2. Internal CSS

Style ditempatkan di dalam tag <style> di bagian <head>:

<head>
    <style>
        p {
            color: blue;
            font-size: 16px;
        }
    </style>
</head>

3. External CSS

Style ditempatkan di file terpisah (.css) dan di-link ke HTML:

<head>
    <link rel="stylesheet" href="styles.css">
</head>

External CSS adalah metode yang paling direkomendasikan karena memisahkan konten dan presentasi.

Selector CSS

Selector Deskripsi Contoh
Element Selector Memilih berdasarkan nama elemen p { color: blue; }
ID Selector Memilih berdasarkan ID (unik) #header { background: gray; }
Class Selector Memilih berdasarkan class .intro { font-weight: bold; }
Universal Selector Memilih semua elemen * { margin: 0; }
Grouping Selector Memilih beberapa elemen h1, h2, h3 { color: purple; }

Box Model

Setiap elemen HTML dianggap sebagai kotak dengan komponen:

  1. Content: Konten aktual (teks, gambar)
  2. Padding: Ruang di dalam border
  3. Border: Garis tepi elemen
  4. Margin: Ruang di luar border
Ini adalah contoh box model dengan padding, border, dan margin

Positioning

Position Deskripsi
static Default (tidak terpengaruh top/bottom/left/right)
relative Relatif terhadap posisi normalnya
absolute Relatif terhadap elemen parent terdekat yang positioned
fixed Relatif terhadap viewport (tetap saat scroll)
sticky Kombinasi relative dan fixed

Flexbox Layout

Flexbox adalah metode layout satu dimensi untuk mengatur item dalam container:

Properti Container:

  • display: flex: Mengaktifkan flexbox
  • flex-direction: Arah item (row/column)
  • justify-content: Perataan horizontal
  • align-items: Perataan vertikal
  • flex-wrap: Wrapping item

Properti Item:

  • flex-grow: Faktor pertumbuhan
  • flex-shrink: Faktor penyusutan
  • flex-basis: Ukuran awal
  • align-self: Override align-items

Grid Layout

CSS Grid adalah metode layout dua dimensi:

Properti Container:

  • display: grid: Mengaktifkan grid
  • grid-template-columns: Definisi kolom
  • grid-template-rows: Definisi baris
  • gap: Jarak antar item
  • grid-template-areas: Template area

Properti Item:

  • grid-column: Posisi kolom
  • grid-row: Posisi baris
  • grid-area: Nama area

Responsive Design

Responsive design membuat tampilan website menyesuaikan berbagai ukuran layar:

Media Queries:

@media (max-width: 768px) {
    body {
        font-size: 14px;
    }
    
    .container {
        width: 100%;
    }
}

Viewport Meta Tag:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Unit Relatif:

  • %: Persentase dari parent
  • vw: 1% dari viewport width
  • vh: 1% dari viewport height
  • rem: Relatif terhadap font-size root
  • em: Relatif terhadap font-size parent

Transisi dan Animasi

Transisi:

.button {
    background: blue;
    transition: background 0.3s ease;
}

.button:hover {
    background: darkblue;
}

Animasi:

@keyframes slide {
    from { transform: translateX(0); }
    to { transform: translateX(100px); }
}

.box {
    animation: slide 2s infinite alternate;
}

Best Practices CSS

  1. Gunakan External CSS: Pisahkan konten dan presentasi
  2. Organisir Kode: Gunakan komentar dan grouping
  3. Mobile-First: Desain untuk mobile dulu, lalu desktop
  4. Optimasi: Minify CSS, gunakan compression
  5. Consistency: Gunakan naming convention (BEM, OOCSS)
  6. Browser Support: Gunakan vendor prefixes dan fallbacks
  7. Performance: Hindari over-qualification, gunakan efficient selectors

Kesimpulan

CSS adalah alat penting untuk:

  • Mengubah tampilan HTML menjadi menarik
  • Membuat layout yang kompleks dengan mudah
  • Membangun desain responsif untuk semua device
  • Menambahkan interaksi dengan animasi dan transisi

Dengan menguasai CSS, Anda dapat mengubah struktur HTML menjadi website yang profesional dan user-friendly.
