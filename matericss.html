1. Pengenalan CSS

CSS (Cascading Style Sheets) adalah bahasa untuk mendesain tampilan halaman web.

CSS digunakan untuk mengatur warna, ukuran, posisi, dan tampilan elemen HTML.

2. Cara Menulis CSS

Inline CSS:

<p style="color: red;">Teks merah</p>

Internal CSS:

<style>
    p { color: red; }
</style>

External CSS:

<link rel="stylesheet" href="style.css">

3. Selector CSS

Element Selector:

p { color: blue; }

Class Selector:

.nama-class { color: green; }

ID Selector:

#nama-id { color: red; }

Universal Selector:

* { margin: 0; padding: 0; }

4. Properti Teks

  • color - warna teks
  • font-size - ukuran font
  • font-family - jenis font
  • font-weight - ketebalan font (bold, normal)
  • text-align - perataan teks (left, center, right)
  • text-decoration - dekorasi teks (underline, none)
  • line-height - jarak antar baris

Contoh:

h1 {
    color: #333;
    font-size: 32px;
    font-family: Arial, sans-serif;
    text-align: center;
}

5. Properti Background

  • background-color - warna latar belakang
  • background-image - gambar latar belakang
  • background-size - ukuran gambar background
  • background-position - posisi gambar background
  • background-repeat - pengulangan gambar (repeat, no-repeat)

Contoh:

body {
    background-color: #f0f0f0;
    background-image: url('gambar.jpg');
    background-size: cover;
}

6. Box Model

Setiap elemen HTML memiliki box model yang terdiri dari:

  • Content - konten elemen
  • Padding - jarak dalam dari konten ke border
  • Border - garis tepi elemen
  • Margin - jarak luar dari border ke elemen lain

Contoh:

.box {
    width: 300px;
    padding: 20px;
    border: 2px solid black;
    margin: 10px;
}

7. Border

div {
    border: 1px solid black;
    border-width: 2px;
    border-style: solid; /* solid, dashed, dotted */
    border-color: red;
    border-radius: 10px; /* sudut melengkung */
}

8. Display Property

  • display: block - elemen mengambil lebar penuh
  • display: inline - elemen sejajar horizontal
  • display: inline-block - kombinasi keduanya
  • display: none - menyembunyikan elemen
  • display: flex - layout flexbox
  • display: grid - layout grid

9. Position

  • static - posisi default
  • relative - relatif terhadap posisi normal
  • absolute - relatif terhadap parent terdekat yang positioned
  • fixed - tetap di layar saat scroll
  • sticky - kombinasi relative dan fixed

Contoh:

.fixed-header {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
}

10. Flexbox

Flexbox untuk membuat layout yang fleksibel:

.container {
    display: flex;
    justify-content: center; /* horizontal alignment */
    align-items: center; /* vertical alignment */
    flex-direction: row; /* row, column */
    gap: 10px;
}

11. Grid

Grid untuk layout dua dimensi:

.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-gap: 20px;
}

12. Responsive Design

Media query untuk membuat tampilan responsif:

/* Mobile */
@media (max-width: 768px) {
    .container {
        width: 100%;
        padding: 10px;
    }
}

/* Desktop */
@media (min-width: 769px) {
    .container {
        width: 800px;
    }
}

13. Pseudo-class

a:hover {
    color: red; /* saat mouse hover */
}

a:active {
    color: blue; /* saat diklik */
}

input:focus {
    border-color: green; /* saat input difokuskan */
}

li:first-child {
    font-weight: bold; /* elemen pertama */
}

li:nth-child(2) {
    color: red; /* elemen ke-2 */
}

14. Transition & Animation

Transition:

button {
    background: blue;
    transition: all 0.3s ease;
}

button:hover {
    background: red;
    transform: scale(1.1);
}

Animation:

@keyframes slide {
    from { left: 0; }
    to { left: 100px; }
}

.box {
    animation: slide 2s infinite;
}

15. Colors

Cara menulis warna di CSS:

  • Nama warna: red, blue, green
  • Hex: #ff0000, #00ff00
  • RGB: rgb(255, 0, 0)
  • RGBA: rgba(255, 0, 0, 0.5) - dengan transparansi
  • HSL: hsl(0, 100%, 50%)

16. Units

  • px - pixel (absolute)
  • % - persen (relative ke parent)
  • em - relative ke font-size parent
  • rem - relative ke font-size root
  • vh - viewport height (1vh = 1% tinggi layar)
  • vw - viewport width (1vw = 1% lebar layar)

17. Contoh Layout Sederhana

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 0 15px;
}

header {
    background: #333;
    color: white;
    padding: 20px 0;
    text-align: center;
}

.content {
    padding: 40px 0;
}

footer {
    background: #333;
    color: white;
    text-align: center;
    padding: 20px 0;
    margin-top: 40px;
}

18. Tips CSS

  • Gunakan class untuk styling yang reusable
  • Hindari inline CSS sebisa mungkin
  • Gunakan CSS reset atau normalize.css
  • Buat nama class yang deskriptif
  • Gunakan shorthand property untuk efisiensi
  • Test di berbagai browser
  • Gunakan tools developer browser (F12)

19. Shorthand Properties

/* Margin */
margin: 10px; /* semua sisi */
margin: 10px 20px; /* atas-bawah kiri-kanan */
margin: 10px 20px 30px 40px; /* atas kanan bawah kiri */

/* Font */
font: bold 16px/1.5 Arial, sans-serif;
/* weight size/line-height family */

/* Background */
background: #fff url('img.jpg') no-repeat center/cover;

/* Border */
border: 2px solid red;
